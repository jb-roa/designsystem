To show the fab sheet, create a FabSheetConfig 

const config: FabSheetConfig = {{ '{' }}
      disabled: 'If fab sheet is enabled/disabled',
      horizontalAlignment: 'Alignment of fab sheet when it opens.',
      actionSheetConfig: {{ '{' }}
        header: 'Your action sheet header',
        subheader: 'Your action sheet sub-header',
        items: [
          {{ '{' }} id: '1', text: 'Option 1' {{ '}' }},
          {{ '{' }} id: '2', text: 'Option 2' {{ '}' }},
          {{ '{' }} id: '3', text: 'Option 3' {{ '}' }},
        ],
      {{ '}' }};
    {{ '}' }};
(Optional) if you need to obtain data back from the fab sheet, you can pass a callback function: (actionSelected)="myCallback($event)" where myCallback can be e.g.: myCallback(selection: ActionSheetItem) {{ '{' }} ... {{ '}' }}

To show the action sheet, create an ActionSheetConfig and pass it to modalController.showActionSheet: const config: ActionSheetConfig={{ '{' }} header: 'Your action sheet header', subheader: 'Your action sheet sub-header', actions: ['Option 1', 'Option 2', 'Option 3'], {{ '}' }}; this.modalController.showActionSheet(config, this.vcRef); (Optional) if you need to obtain data back from the action sheet, you can pass a callback function: this.modalController.showActionSheet(config, this.vcRef, myCallback); where myCallback can be e.g.: myCallback(selection: string) {{ '{' }} ... {{ '}' }} We can use modalController.hideTopmost to hide (and destroy its component) the topmost action sheet. We can pass an optional result parameter to it, which is then going to be available inside the callback

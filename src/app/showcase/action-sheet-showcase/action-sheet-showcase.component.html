--- Inside the parent(caller) component ---
To show the action sheet, create an ActionSheetConfig and pass it to modalService.showActionSheet:

const config: ActionSheetConfig = {{ '{' }}
   header: 'Your action sheet header',
   subheader: 'Your action sheet sub-header',
   actions: ['Option 1', 'Option 2', 'Option 3'],
{{ '}' }};
this.modalService.showActionSheetWindow(config, this.vcRef);

(Optional) if you need to obtain data back from the action sheet, you can pass a callback function:

this.modalService.showActionSheetWindow(config, this.vcRef, myCallback);
where myCallback can be e.g.:

myCallback(selection: string) {{ '{' }}
   ...
{{ '}' }}

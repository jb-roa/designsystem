To show the action sheet, create an ActionSheetConfig and pass it to modalController.openActionSheet: const config: ActionSheetConfig={{ '{' }} header: 'Your action sheet header', subheader: 'Your action sheet sub-header', actions: ['Option 1', 'Option 2', 'Option 3'], {{ '}' }}; this.modalController.openActionSheet(config, this.vcRef); (Optional) if you need to obtain data back from the action sheet, you can pass a callback function: this.modalController.openActionSheet(config, this.vcRef, myCallback); where myCallback can be e.g.: myCallback(selection: string) {{ '{' }} ... {{ '}' }}

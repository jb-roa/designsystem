To show the fab sheet, create an FabSheetConfig 

const config: FabSheetConfig = {{ '{' }}
      disabled: 'If fab sheet is enabled/disabled',
      horizontalAlignment: 'Alignment of fab sheet when it opens.',
      actionSheetConfig: {{ '{' }}
        header: 'Your action sheet header',
        subheader: 'Your action sheet sub-header',
        actions: ['Option 1', 'Option 2', 'Option 3'],
      {{ '}' }};
    {{ '}' }};
(Optional) if you need to obtain data back from the fab sheet, you can pass a callback function: (actionSelected)="myCallback($event)" where myCallback can be e.g.: myCallback(selection: string) {{ '{' }} ... {{ '}' }}
